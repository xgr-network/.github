# XGR Network

XGR Network builds XGRChain and XDaLa, an EVM compatible blockchain and process engine for deterministic on chain validation, orchestration and execution.

## What we build
- XGRChain: EVM compatible blockchain focused on process execution
- XDaLa: Validation to execution engine for auditable workflows
- XRC: Public standards for rules, orchestration and execution
