## What is XDaLa?

**XDaLa** turns rules, external data, encryption and smart-contract execution into auditable on-chain sessions.

Instead of treating blockchain transactions as isolated transfers, XDaLa enables deterministic multi-step workflows with validation, orchestration and execution.

It is designed for use cases that require:

- verifiable process execution
- external data integration
- encrypted inputs and outputs
- auditability
- compliance-sensitive workflows
- controlled smart-contract execution
